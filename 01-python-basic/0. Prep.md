# Day 0 – Cài đặt môi trường Python Automation

## 1. Cài Python 3.x + pip

1. Vào trang: [https://www.python.org/downloads/](https://www.python.org/downloads/)
2. Tải bản mới nhất Python 3.x (>= 3.10).
3. Khi chạy file cài đặt NHỚ tick:

   * **Add Python to PATH**
4. Kiểm tra sau khi cài:

```bash
python --version
pip --version
```

Nếu đều in ra version → OK.

---

## 2. Tạo folder dự án & Virtual Environment (venv)

1. Tạo thư mục, ví dụ:

```text
C:\Automation\PlaywrightPython
```

2. Mở terminal trong thư mục đó.
3. Tạo môi trường ảo:

```bash
python -m venv venv
```

4. Kích hoạt venv (Windows):

```bash
venv\Scripts\activate
```

Thấy prompt dạng:

```text
(venv) C:\Automation\PlaywrightPython>
```

là đúng.

---

## 3. Cài Visual Studio Code + Extensions

1. Tải VS Code: [https://code.visualstudio.com/](https://code.visualstudio.com/)
2. Cài các extension tối thiểu:

   * **Python (Microsoft)**
   * **Pylance**
   * (Tuỳ chọn) Playwright Test for VS Code

---

## 4. Tạo và dùng `requirements.txt`

Trong thư mục project, tạo file:

```text
requirements.txt
```

Nội dung khởi đầu:

```text
pytest
playwright
pytest-playwright
```

Sau này có thể thêm:

```text
requests
python-dotenv
```

Cài thư viện:

```bash
pip install -r requirements.txt
```

---

## 5. Cài browser cho Playwright

Sau khi cài xong package `playwright`:

```bash
playwright install
```

Lệnh này sẽ tải Chromium/Firefox/WebKit về.

---

## 6. Kiểm tra Pytest chạy OK

Tạo file `test_sample.py`:

```python
def test_example():
    assert 1 == 1
```

Chạy:

```bash
pytest -v
```

Nếu test chạy pass → Pytest OK.

---

## 7. Kiểm tra Playwright chạy OK

Thử chạy Playwright với Chromium:

```bash
pytest --browser chromium --headed
```

Hoặc mở codegen demo:

```bash
playwright codegen https://playwright.dev
```

Nếu browser bật lên và không lỗi → môi trường Playwright OK.
